# TG/VK Post Planner - Image Repository

Репозиторий для хранения изображений, используемых в постах Telegram и VK.

## Структура

```
images/
├── events/          # Изображения для событий
│   ├── 2026/
│   │   ├── 01/      # Январь 2026
│   │   ├── 02/      # Февраль 2026
│   │   └── ...
├── templates/       # Шаблоны изображений
│   └── ...
└── thumbnails/      # Миниатюры
    └── ...
```

## Использование

Изображения доступны через GitHub Pages:
- Базовый URL: `https://nickmed1988.github.io/tg-vk-poster-images`
- Пример: `https://nickmed1988.github.io/tg-vk-poster-images/images/events/2026/01/image.jpg`

## Загрузка изображений

Изображения загружаются автоматически через планировщик постов или вручную через GitHub API.
